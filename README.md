# vue-blog-express

使用 express 搭建个人博客系统的后端

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
set DEBUG='vue-blog-express':* & npm run devstart
```

## 说明

在mysql中创建 vueblog 数据库，然后在 vueblog 数据库中执行项目中的 vueblog.sql 文件，创建需要的表。
